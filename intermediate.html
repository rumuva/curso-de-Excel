Nivel Intermedio

Este nivel está pensado para usuarios que ya dominan las nociones básicas y desean ampliar sus habilidades. Aprenderás a transformar datos y a utilizar herramientas de análisis más potentes.

Temas incluidos

  • Funciones avanzadas: VLOOKUP, INDEX y MATCH, así como las funciones condicionales IF e IFS.
  • Tablas y gráficos dinámicos: crea, configura y analiza tablas dinámicas y gráficos dinámicos para explorar tus datos en profundidad.
  • Validación y formato condicional: utiliza la validación de datos junto con reglas de formato condicional para asegurar la calidad de la información.
  • Macros básicas: introduce la automatización mediante el grabador de macros y aprende a ejecutar tareas repetitivas.

Al concluir el nivel intermedio, serás capaz de manejar datos más complejos y de crear informes interactivos que faciliten la toma de decisiones.
